# Chess!

Right click on index.html file in your local machine's file explorer, and select the open in browser option. 
Ensure that the browser that you open the file in is Chrome. 
Enjoy Chess: Odyssey!

If you encounter an error in which the pieces are not moving properly, or two pieces appear in the same square after a capture, refresh your page.

Tip: when dragging pieces, try your best to drop them directly on top of the opposing player's piece, not the surrounding space of the square.
